# Belle Stetic Website

Este es el sitio web oficial para **Belle Stetic**, desarrollado con HTML5, CSS3 y JavaScript para ofrecer un alto rendimiento, seguridad y un diseño moderno y adaptable.

## Cómo ver el sitio
Simplemente abre el archivo `index.html` en tu navegador web favorito (Chrome, Firefox, Edge, etc.).

## Estructura del Proyecto

- **index.html**: Página de inicio con servicios destacados.
- **nosotros.html**: Información sobre la historia, valores y equipo.
- **contacto.html**: Formulario de contacto y ubicación.
- **turismo-estetico.html**: Landing page especializada para turismo estético.
- **css/style.css**: Hoja de estilos principal (diseño, colores, tipografía).
- **js/main.js**: Scripts para el menú móvil y efectos de desplazamiento.

## Personalización

### Cambiar Colores
Puedes modificar los colores principales editando las variables CSS al inicio de `css/style.css`:
```css
:root {
    --primary-color: #D4AF37; /* Color principal */
    --secondary-color: #2C3E50; /* Color secundario */
}
```

### Formulario de Contacto
El formulario en `contacto.html` es visual. Para recibir correos electrónicos, necesitarás conectar un servicio de backend o usar una solución como [Formspree](https://formspree.io/).
